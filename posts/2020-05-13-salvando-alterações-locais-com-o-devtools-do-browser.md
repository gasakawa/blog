Muitas vezes na hora de desenvolver um site ou uma página web, nos deparamos com a necessidade de testar algumas mudanças em propriedades CSS para validar se o efeito visual é o desejado.

Para isto utilizamos o **DevTools** no [Google Chrome](https://developers.google.com/web/tools/chrome-devtools) ou do [Mozilla Firefox](https://developer.mozilla.org/pt-BR/docs/Tools).
